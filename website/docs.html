Quick Start

Get your first token launched in under 2 minutes.

Step 1: Connect Wallet

Connect using Phantom, Solflare, or create a wallet with Privy using just your email or social login.

Step 2: Configure Token

Fill in your token details:

  • Name - Your token's display name
  • Symbol - Short ticker (e.g., LAUNCHR)
  • Description - Tell the story behind your token
  • Image - Upload a logo (PNG/JPG, 500x500 recommended)
  • Banner - Optional header image for your token page

Step 3: Add Socials

Link your Twitter, Telegram, Website, and Discord to build community trust.

Step 4: Set Dev Buy (Optional)

Purchase tokens at launch with 0.01-5 SOL. This becomes your initial position.

Step 5: Launch

Sign the transaction and your token goes live on Pump.fun instantly.

Creator Fees Explained

Pump.fun charges 1% on every trade. Of this:

  • 0.5% goes to Pump.fun as platform fee
  • 0.5% goes to the token creator (you!)

LAUNCHR's allocation engine lets you programmatically route this 0.5% creator fee to maximize token value.

Fee Structure

Creator fees collected from trading are split between the LAUNCHR ecosystem and your personal allocation engine.

Fee Type Amount Destination
LAUNCHR Holder Fee 1% LAUNCHR holder reward pool
Creator Allocation 99% Your allocation engine

The 1% LAUNCHR holder fee creates sustainable value for $LAUNCHR token holders while you retain 99% control over your creator revenue.

Your 99% creator share is distributed across four strategic channels that you control.

Vanity Addresses

Launch tokens with memorable contract addresses that end in custom suffixes.

How It Works

LAUNCHR pre-generates keypairs with addresses matching specific patterns. When you launch, we can assign a vanity address to your token.

// Example vanity addresses
"7xK9...lchr"   // Ends in "lchr"
"4mN2...pump"   // Ends in "pump"
"8bQ1...moon"   // Ends in "moon"

Generation Time

Pattern Length Probability Avg Time (8 cores)
3 chars 1 in 39,304 ~0.4 seconds
4 chars 1 in 1.3M ~13 seconds
5 chars 1 in 45M ~8 minutes
6 chars 1 in 1.5B ~4 hours
‚ÑπÔ∏è Base58 Alphabet

Solana addresses use Base58 which excludes 0, O, I, and l to avoid confusion. Valid characters: 123456789ABCDEFGHJKLMNPQRSTUVWXYZabcdefghijkmnopqrstuvwxyz

Token Creation

LAUNCHR creates tokens on Pump.fun's infrastructure using PumpPortal API.

Creation Flow

  1. Upload metadata (image, banner) to IPFS
  2. Generate token keypair (or use vanity address)
  3. Build create transaction with metadata
  4. User signs with connected wallet
  5. Transaction submitted to Solana
// Token creation parameters
{
  "name": "My Token",
  "symbol": "MTK",
  "description": "The best token ever",
  "twitter": "https://twitter.com/mytoken",
  "telegram": "https://t.me/mytoken",
  "website": "https://mytoken.xyz",
  "initialBuyAmount": 0.5  // SOL for dev buy
}

Security

LAUNCHR implements multiple security layers to protect users and funds.

5-Layer Mission Control Security

  1. Wallet Verification - All transactions require explicit user signature
  2. Non-Custodial - We never hold your private keys or funds
  3. On-Chain Transparency - All operations verifiable on Solana explorer
  4. Rate Limiting - API protection against abuse
  5. Input Validation - All user inputs sanitized and validated
üö® Security Notice

Never share your private key or seed phrase. LAUNCHR will never ask for these. Always verify transaction details before signing.

Smart Contract Security

Token creation uses Pump.fun's audited contracts. Fee distribution operates through verified on-chain programs.
